API Documentation

Introduction:

This API provides various endpoints to interact with the E-commerce system.

Endpoints:

  1. POST /products/create

    • Open Postman >> Go to Request
    • Paster this URL >> http://127.0.0.1:5000/products/create
    • Choose the method as "POST" and Go to Body
    • Select Raw >> Add the product keys and values
    • Send request
    • Product is Added Successfully.

  2. GET /products

    • It shows a list of all products.

  3. GET /products/{id}

    • It shows a details of a specific product.

  4. POST /cart

    • Again go to Postman.
    • Select POST method
    • Paster URL >> http://127.0.0.1:5000/cart
    • Select Raw >> Add the product id and quantity
    • Send request
    • Product is added to the cart Successfully.

  5. GET /cart

    • Shows the items which is in the Cart.

  6. DELETE /cart/{id}

    • Go to Postman.
    • Paster URL >> http://127.0.0.1:5000/cart/3
    • Choose method as DELETE
    • Send Request
    • Item is removed from the cart Successfully.

Usage:

  • Ensure to include necessary parameters in the request body.
  • Use appropriate HTTP methods for different actions.
